## 7. 잘한 점 (살릴 강점)

✅ **우선순위가 명확함**
- 뉴스 요약 + 시장 분석이 핵심 (이메일은 부가)
- 웹사이트를 블로그처럼 표시하는 구체적 이미지 있음
- → 개발 방향이 흔들리지 않을 것

✅ **환경 제약이 없음**
- API 호출, 라이브러리 설치 자유
- 기술적 막힘이 없음
- → Claude Code + 외부 도구로 빠르게 해결 가능

✅ **배포를 선택사항으로**
- 로컬 작동만 해도 7/15 시연 가능
- DB/서버 설정 복잡도 낮춤
- → 3주 안에 충분히 가능

### 🛠️ 기술 선택 제안

| 항목 | 추천 | 이유 |
|------|------|------|
| **백엔드** | **Node.js/Express** ⭐ | Python은 학습곡선. 웹앱 경험 활용. 더 빠름 |
