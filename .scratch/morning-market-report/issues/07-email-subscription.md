# 이메일 구독 폼 & 미리보기

Status: ready-for-agent

## 구축할 내용

웹사이트에 이메일 구독 신청 폼을 추가하고, 구독자가 받을 이메일의 모형/미리보기를 표시합니다. v1에서는 이메일 저장만 하고 실제 발송은 v2에서 구현합니다. 사용자가 "이렇게 매일 받을 이메일" 형태를 미리 볼 수 있도록 합니다.

## 수락 기준

- [ ] 웹사이트에 이메일 구독 폼 추가 (이름, 이메일 입력)
- [ ] `POST /api/subscription/signup` 엔드포인트 구현
- [ ] 구독자 정보를 `email_subscribers` 테이블에 저장
- [ ] 이메일 미리보기 화면 구현 (HTML 형태로 "이렇게 받을 거예요" 표시)
- [ ] 미리보기에 뉴스 5개 요약, 시장 전망, 투자포인트 포함
- [ ] 구독 폼 제출 시 성공 메시지 표시
- [ ] 실제 폼 제출 및 저장 동작 확인

## 선행 조건

None - 병렬로 진행 가능

## 기술 노트

**API 엔드포인트:**
```
POST /api/subscription/signup
Body: {
  "email": "string",
  "name": "string"
}
Response: {
  "status": "success",
  "subscription_id": "string",
  "message": "구독이 신청되었습니다"
}
```

**이메일 미리보기 HTML 구조:**
```
────────────────────────────────
📰 아침 시장 전망 보고서
────────────────────────────────
면책 고지: 투자는 본인 판단으로

시장 전망:
[1문단]

투자포인트:
• [포인트 1]
• [포인트 2]
• [포인트 3]

오늘의 뉴스:
[뉴스 5개 요약]

원문 보기: [링크들]
────────────────────────────────
```

**주의:**
- v1에서는 실제 이메일 발송 금지
- 저장된 이메일로는 아직 아무것도 발송하지 않음
- v2에서 APScheduler + 이메일 발송 구현 예정
