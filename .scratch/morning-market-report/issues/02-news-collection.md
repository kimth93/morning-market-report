## 구축할 내용

네이버뉴스 API에서 경제·증권 카테고리의 최신 뉴스 5개를 실시간으로 수집합니다. 각 뉴스 항목에서 제목, 언론사, 발행일, 원문 링크를 추출하여 데이터베이스에 저장하고, 후속 요약 모듈이 처리할 수 있도록 구조화합니다.

## 수락 기준

- [ ] 네이버뉴스 API 연동 완료 (또는 대체 뉴스소스)
- [ ] 경제·증권 카테고리에서 뉴스 Top 5 수집
- [ ] 각 뉴스에서 필수 정보 추출: 제목, 언론사, 발행일, 링크
- [ ] 수집한 뉴스를 `news_items` 테이블에 저장
- [ ] 중복 뉴스 제외 로직 구현
- [ ] 실제 뉴스 5개를 수집하여 정상 작동 확인

## 기술 노트

**Naver News API 대체안:**
- Naver News 웹스크래핑 (requests + BeautifulSoup)
- RSS 피드 파싱
- 기타 공개 뉴스 API

**출력 데이터 구조:**
```json
{
  "news_items": [
    {
      "title": "string",
      "source": "string",
      "date": "2026-06-24",
      "url": "string",
      "collected_at": "2026-06-24T10:30:00Z"
    }
  ]
}
```
