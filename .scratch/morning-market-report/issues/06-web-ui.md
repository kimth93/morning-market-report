## 구축할 내용

React 기반 웹 인터페이스를 구축하여 뉴스 요약, 출처, 시장 전망, 투자포인트를 사용자 친화적으로 표시합니다. "지금 생성" 버튼을 눌러 실시간으로 최신 요약을 생성할 수 있도록 하고, 법적 면책 고지를 명확하게 표시합니다.

## 수락 기준

- [ ] React 앱 초기 설정 (Create React App 또는 Vite)
- [ ] "지금 생성" 버튼이 `/api/news/generate` 호출
- [ ] 뉴스 5개 요약 표시 (제목, 요약 텍스트)
- [ ] 각 뉴스에 출처 정보 명시 (언론사, 발행일, 원문 링크)
- [ ] 할루시네이션 위험도 표시 (low/medium/high)
- [ ] 시장 전망 1문단 표시
- [ ] 투자포인트 3개 표시
- [ ] "투자는 본인의 판단으로 하시기 바랍니다" 면책 고지 상단에 명확히 표시
- [ ] 기본 가독성 확보 (반응형 레이아웃 선택사항)
- [ ] 실제 브라우저에서 "지금 생성" 버튼 동작 확인

## 기술 노트

**UI 구조:**
```
┌─────────────────────────────────────┐
│ [면책 고지: 투자는 본인 판단으로]      │
├─────────────────────────────────────┤
│ 시장 전망: [1문단]                   │
├─────────────────────────────────────┤
│ 투자포인트:                          │
│  • [포인트 1]                        │
│  • [포인트 2]                        │
│  • [포인트 3]                        │
├─────────────────────────────────────┤
│ [뉴스 1]                             │
│ 요약: [1-2문단]                      │
│ 출처: [언론사] | [날짜] | [링크]     │
│ 위험도: [low|medium|high]            │
├─────────────────────────────────────┤
│ ... (뉴스 2-5)                       │
├─────────────────────────────────────┤
│ [지금 생성 버튼]                      │
└─────────────────────────────────────┘
```

- react
- axios (또는 fetch)
