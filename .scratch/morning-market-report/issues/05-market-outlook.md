Status: ready-for-agent

## 부모 이슈

[#3 Claude 요약 + 신뢰성 제어](./03-claude-summarization.md)

## 구축할 내용

수집된 뉴스 5개의 요약을 바탕으로 당일 시장 전망 1문단과 투자포인트 3개를 생성합니다. 신뢰성 규율을 동일하게 적용하여 할루시네이션이나 과도한 투자 권유가 없도록 제어합니다. 프로토타입 수준의 완성도도 허용합니다.

## 수락 기준

- [ ] 시장 전망 생성 (1문단, 조건부 표현 사용)
- [ ] 투자포인트 3개 생성 (각 1-2줄)
- [ ] 할루시네이션 금지 및 단정성 제어 적용
- [ ] 편향성 없음 확인 (특정 섹터 과도 강조 제외)
- [ ] `/api/news/generate` 응답에 포함 가능하도록 통합
- [ ] 실제 뉴스로 생성 후 품질 검증

## 기술 노트

**API 응답에 추가되는 필드:**
```json
{
  "market_outlook": "string (1문단)",
  "investment_points": [
    "string",
    "string",
    "string"
  ]
}
```

**생성 프롬프트 원칙:**
- 뉴스 5개 요약을 기반으로만 작성
- "~할 가능성이 있다", "주목할 만하다" 등 조건부 표현 사용
- 특정 종목이나 섹터 과도 강조 금지
